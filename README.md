# Openmobo Notes Program
Markdown editor using electron, nodejs

## Installation and debuggging

**debugging**
 - [ ] install nodejs
 - [ ] clone this repo
 - [ ] in the cli at this repo folder type `npm install`
 - [ ] in the cli at this folder type `electron .`
